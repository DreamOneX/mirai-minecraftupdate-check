# MinecraftUpdateChecker

一个用来检测Minecraft相关更新的插件 **(WIP)**

## 命令

### `checkver minecraft <CheckType>`

检查Minecraft的最新版

### `subupdate minecraft <CheckType>`

订阅更新

## CheckType

1. MC_RELEASE
2. MC_SNAPSHOT
